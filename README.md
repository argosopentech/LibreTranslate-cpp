# LibreTranslate-cpp
[LibreTranslate](https://libretranslate.com) C++ bindings.

## Dependencies
[libcurl](https://curl.se/libcurl/c/http-post.html)
```
sudo apt-get install -y libcurl4-openssl-dev
curl-config --libs
```

## Building
```
make
```

## Running
```
./libretranslate
```

## Clean
```
make clean
```
