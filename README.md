# LibreTranslate-cpp
[LibreTranslate](https://libretranslate.com) C++ bindings.

## Dependencies
[Boost Asio](https://www.boost.org/doc/libs/1_77_0/doc/html/boost_asio.html)
```
sudo apt-get install -y libasio-dev
```

## Building
```
make
```

## Running
```
./libretranslate
```

## Clean
```
make clean
```
